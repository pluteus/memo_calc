memo_calc はメモ帳と電卓を一枚にしたノートです。左に書いた数式は右の列に即座に計算され、行番号で参照しあい、増減すれば自動で追従します。変数も関数呼び出しも要りません。

FEATURES

罫線の右側で、計算がついてくる。

live calc

入力した瞬間に右列で計算

数式を書くだけで、右側の結果カラムにリアルタイムで答えが出ます。180000 * 10% のようなパーセント記法もそのまま解釈します。

row refs

行を S7 のように参照

行番号タップで S7 を挿入。行番号を縦にスワイプすれば sum(S7,S11) のような範囲合計を一発で入力できます。

auto-shift

行を増減しても参照はズレない

途中に行を足したり消したりしても、S 参照は自動的に振り直され、参照先の実体は常に保たれます。

notes

日本語のラベル行はコメント扱い

数字や演算子を含まない行、全角文字を含む行は自動でコメントとして無視されるので、見出しやメモを自由に挟めます。

display

小数 / 切捨 / 切上 / 四捨五入

結果の見せ方をワンタップで切り替え。計算自体は常に元の小数値のまま、表示形式だけが変わります。

offline

PWA・自動保存・元に戻す

ホーム画面に追加してオフラインでも利用可能。編集内容は自動保存され、アプリを閉じても次回そのまま復元。Undo / Redo にも対応しています。

EXAMPLE

見積もりを書くだけで、合計まで出る。

# 見積もり 材料費 84000 作業費 36000 # 合計（3行目〜6行目） sum(S3,S6)
　 　 　 84,000 　 　 36,000 　 　 120,000
HOW IT WORKS

使い方は、ただ書くだけ。

01

ノートに数式や数字を書く

計算式でも、ラベルや見出しでも自由に書けます。数字を含まない行は自動でコメント扱いになります。

02

右列の結果を確認する

該当する行の右側に、リアルタイムで計算結果が表示されます。

03

行番号で他の行を参照する

行番号をタップして S 参照を挿入、縦スワイプで範囲合計。書き足すたびに参照は自動で追従します。

次のメモは、計算もしてくれる。
